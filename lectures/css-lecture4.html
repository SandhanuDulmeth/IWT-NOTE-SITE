Essential HTML & CSS Reference

A quick-reference guide covering essential HTML semantic tags, CSS visual & text styling, spacing & sizing properties, and layout & interaction properties.

01 Essential HTML Tags

Lecture 4 ¬∑ SCS 1311 ¬∑ Topics: <main>, <nav>, meta viewport, <iframe>

üìÑ <main>

Holds the primary content of the page ‚Äî content unique to this page (not headers, footers, or navigation).

html 
<main>Main content</main>

üß≠ <nav>

Contains navigation links ‚Äî menus, breadcrumbs, table of contents.

html 
<nav>Menu</nav>

üì± Meta Viewport

Enables proper mobile scaling ‚Äî the page adapts to the device width instead of rendering at desktop size.

html 
<meta name="viewport" content="width=device-width, initial-scale=1.0">

ü™ü <iframe>

Embeds another webpage inside the current page ‚Äî maps, videos, external content.

html 
<iframe src="page.html"></iframe>

üèãÔ∏è Practice

  1. Wrap your page content in a <main> tag
  2. Create a <nav> with three links
  3. Add the meta viewport tag and test on a mobile emulator
  4. Embed a YouTube video using <iframe>

04 Layout & Interaction

üìã Quick Reference

Property What it does Example value
position Controls positioning mode absolute
top / left Moves a positioned element 10px
z-index Stacking order (top/bottom) 2
overflow Handles extra content hidden
:hover Style on mouse hover color: red
cursor Changes mouse pointer pointer
box-sizing Controls size calculation border-box

üìå position

Controls how an element is placed in the document flow.

  • static ‚Äî default, normal flow
  • relative ‚Äî offset from normal position
  • absolute ‚Äî placed relative to nearest positioned ancestor
  • fixed ‚Äî stays fixed during scroll
  • sticky ‚Äî switch between relative and fixed
css ‚Äî position 
.badge {
  position: absolute;
  top: 10px;
  left: 10px;
}

üìö z-index

Controls stacking order ‚Äî which element appears on top when they overlap. Higher value = on top. Only works on positioned elements.

css ‚Äî z-index 
.box1 {
  position: absolute;
  z-index: 1;
}

.box2 {
  position: absolute;
  left: 10px;
  top: 10px;
  z-index: 2;   /* appears on top */
}
css ‚Äî overflow 
.box {
  overflow: hidden;
}

üñ±Ô∏è :hover Pseudo-class

Applies styles when the mouse hovers over an element.

css ‚Äî :hover 
a:hover {
  color: red;
}
css ‚Äî cursor 
button {
  cursor: pointer;
}

üì¶ box-sizing

Controls how width and height are calculated:

  • content-box (default) ‚Äî width/height = content only; padding & border added on top
  • border-box ‚Äî width/height includes padding & border; keeps element size fixed to the set value
css ‚Äî box-sizing 
* {
  box-sizing: border-box;
}

üèãÔ∏è Practice

  1. Position a badge absolute inside a relative container
  2. Stack two overlapping boxes using z-index
  3. Add :hover effects to links and buttons
  4. Apply * { box-sizing: border-box; } globally and compare element sizes before/after
