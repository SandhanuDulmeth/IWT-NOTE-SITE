Introduction to CSS

What CSS is, why we use it, how to apply it to HTML, CSS syntax, selectors, combinators, and the philosophy of separation of concerns.

01 What is CSS & Why Use It

Lecture 1 ¬∑ SCS 1311 ¬∑ Topics: CSS definition, purposes, benefits

üé® What is CSS?

  • CSS (Cascading Style Sheets) ‚Äî a stylesheet language used to describe the presentation of documents written in HTML or XML
  • Styles HTML elements
  • Controls: colors, fonts, spacing, layout
  • Keeps content (HTML) and design (CSS) separate
  • Recommended by W3C

‚úÖ Why Use CSS?

  • Improves website appearance
  • Makes pages consistent
  • Easier maintenance and updates
  • One CSS file can style multiple pages

üí° Core Concept ‚Äî Separation of Concerns

Think of HTML as the skeleton of a building and CSS as the paint, lighting, and interior design.

  • Without CSS, the page works. With CSS, the page feels alive.
  • HTML = structure
  • CSS = presentation
  • JavaScript = behavior

That separation is one of the quiet philosophical victories of web engineering ‚Äî like dividing physics into mechanics, electromagnetism, and thermodynamics so each can evolve without breaking the others.

02 Ways to Apply CSS

1Ô∏è‚É£ Inline CSS

  • CSS written inside an HTML element using the style attribute
  • Affects only that element
  • Used for quick tests or small changes
  • Multiple styles separated with semicolons
html 
<p style="color: blue; font-size: 18px;">Hello</p>

2Ô∏è‚É£ Internal (Embedded) CSS

  • Written inside <style> tag in the <head>
  • Affects only one HTML page
  • Good for single-page styling
html 
<html>
<head>
<style>
p {
  color: green;
  font-size: 18px;
}
</style>
</head>
<body>
  <p>Hello</p>
</body>
</html>

3Ô∏è‚É£ External CSS

  • CSS stored in a separate .css file
  • Linked to HTML using <link>
  • Reusable across multiple pages
  • Easier maintenance ‚Äî change one file, update all pages
html + css 
<!-- HTML file -->
<link rel="stylesheet" href="style.css">

/* style.css */
p {
  color: red;
}

üì• @import Method

  • Another way to include external CSS
  • Written inside <style> or a CSS file
html 
<style>
@import url("style.css");
</style>

üèãÔ∏è Practice

  1. Style a paragraph using inline CSS (change color and font size)
  2. Create internal CSS in <style> to style all headings
  3. Create an external style.css file and link it to your HTML page
  4. Try the @import method to include a second CSS file

03 CSS Syntax & Selectors

üìù CSS Syntax

Basic structure of a CSS rule:

selector {
  property: value;
}
css 
h1 {
  color: blue;
  font-size: 24px;
}

üéØ CSS Selectors ‚Äî Five Categories

  1. Simple selectors ‚Äî element, class, ID, universal, group
  2. Combinator selectors ‚Äî descendant, child, adjacent sibling, general sibling
  3. Pseudo-class selectors ‚Äî :hover, :focus, :nth-child()
  4. Pseudo-element selectors ‚Äî ::before, ::after, ::first-line
  5. Attribute selectors ‚Äî [type="text"], [href^="https"]

üî§ Simple Selectors

Selector Syntax Selects
Element h1 All <h1> elements
Class .name Elements with class="name"
ID #name Element with unique id="name"
Universal * All elements on the page
Group h1, p Multiple selectors at once
css 
/* Element selector */
h1 { color: red; }

/* Class selector */
.first-paragraph { background-color: green; }

/* ID selector */
#second-paragraph { color: red; }

/* Universal selector */
* { color: green; }

/* Group selector */
h1, p { color: blue; }

üèãÔ∏è Practice

  1. Use an element selector to change all <p> text to green
  2. Create a class .highlight and apply it to two different elements
  3. Use an ID selector to uniquely style one heading
  4. Use the universal selector to set a default font family
  5. Group h1, h2, h3 to share the same color

04 Combinator Selectors

üîó Combinator Selectors

Select elements based on their relationship to other elements.

Combinator Symbol Selects
Descendant space All nested elements (children, grandchildren, etc.)
Child > Only direct children
Adjacent sibling + Element immediately after another
General sibling ~ All siblings after an element
css 
/* Descendant ‚Äî all p inside div */
div p { color: green; }

/* Child ‚Äî only direct p children of div */
div > p { color: blue; }

/* Adjacent sibling ‚Äî first p right after h2 */
h2 + p { color: red; }

/* General sibling ‚Äî all p after h2 */
h2 ~ p { color: purple; }

üèãÔ∏è Practice

  1. Use the descendant selector to style only paragraphs inside a specific <div>
  2. Use the child selector to style only direct children of a container
  3. Use h2 + p to style only the first paragraph after each heading
  4. Use h2 ~ p and compare the difference to the adjacent sibling

05 Styling div & span

üì¶ Styling a <div>

  • <div> is a block-level container
  • Used to group elements
  • No visual effect by default ‚Äî styled using CSS
css 
div {
  border: 1px solid black;
  padding: 10px;
}

üè∑Ô∏è The <span> Element

  • Inline container ‚Äî does not start a new line
  • Used to style part of text within a block
  • No visual effect by default
  • Often used with classes
html + css 
<!-- HTML -->
<p>This is <span class="highlight">important</span> information.</p>

/* CSS */
.highlight {
  color: red;
  font-weight: bold;
}

‚öñÔ∏è <div> vs <span>

Element Type Purpose
<div> Block-level Groups sections
<span> Inline Styles part of text

üèãÔ∏è Practice

  1. Create a <div> with a border, padding, and background color
  2. Use <span> with a class to highlight a keyword within a paragraph
  3. Build a card layout using <div> for the container and <span> for inline labels
