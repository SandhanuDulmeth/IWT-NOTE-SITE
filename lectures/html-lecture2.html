03 HTML Attributes

Lecture 2 (continued) ¬∑ Topics: Attribute syntax, common attributes, global/local/boolean types

üìù What Are Attributes?

  • Provide extra information about an element
  • Written inside the opening tag
  • Format: attribute="value"

üñºÔ∏è Image Attributes

Attribute Purpose
src Image source (file path or URL)
width Width of image
height Height of image
alt Image description (accessibility)

üîó Link & Input Attributes

Attribute Purpose
href Link target URL
type Input type (text, password, email‚Ä¶)
value Input default value

üîñ Types of Attributes

1. Global Attributes ‚Äî can be used on any element

  • id ‚Äî unique identifier (must be unique on page)
  • class ‚Äî reusable group name for styling
  • title ‚Äî tooltip text on hover

2. Local Attributes ‚Äî specific to certain elements

  • src, alt (for <img>)
  • href (for <a>)

3. Boolean Attributes ‚Äî present or absent, no value required

  • disabled ‚Äî disables the element
  • required ‚Äî makes input mandatory
  • autofocus ‚Äî focuses element on page load
html 
<!-- Image with attributes -->
<img src="cat.jpg" width="300" height="200" alt="A cute cat">

<!-- Link -->
<a href="https://example.com">Go</a>

<!-- Input types -->
<input type="password">
<input type="text" value="Hello">

<!-- Boolean attributes -->
<input type="text" required autofocus>
<button disabled>Can't click</button>

<!-- Global attributes -->
<div id="main" class="container">
  <p title="Tooltip text">Hover me</p>
</div>

  3. Build a form with required, disabled, and autofocus boolean attributes
  4. Use id, class, and title global attributes on various elements
