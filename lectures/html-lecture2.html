Elements, Tags, Text & Attributes

HTML elements, tags, text formatting, headings, paragraphs, and attribute types.

01 Elements, Tags & Markup

‚ùì What is HTML?

  • HTML stands for Hypertext Markup Language.
  • It is the standard language used to create and design the structure of a web page.
  • It helps organized content, define structure, and create relationships between elements.
  • HTML uses markup to annotate text, images, and other content.

üè∑Ô∏è Tags and Elements

Tags: Keywords inside angle brackets < >. They tell the browser how to display content.

Element: Consists of the Opening Tag + Content + Closing Tag.


Syntax of a Tag
<tagname>content</tagname>
Example Element
<p>Hello World</p>

Types of Tags/Elements

1. Paired Elements

Have both an opening and a closing tag.

  • <p>Text</p>
  • <h1>Heading</h1>
2. Self-closing (Void) Elements

Have no closing tag and no content.

  • <br> (Line break)
  • <hr> (Horizontal rule)
  • <img> (Image)
  • <input> (Input field)

üîÑ Nesting Elements

Elements can go inside each other. Proper nesting is crucial.

Correct: <p>This is <strong>important</strong> text.</p>

Incorrect: <p>This is <strong>important</p></strong>

02 Document Structure

Basic HTML Structure 
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, world!</h1>
    <p>This is my first web page.</p>
</body>
</html>

üèóÔ∏è Roles of Each Section

  • <!DOCTYPE html>: Declares the document type as HTML5. Prevents "quirks mode".
  • <html>: The root element containing the entire document.
  • <head>: Contains metadata (not visible on the page). Includes title, CSS links, fonts, etc.
  • <body>: Contains everything visible to the user (headings, text, images, etc.).

üí¨ HTML Comments

Notes inside code, not displayed in browsers. used for explanations or disabling code.

<!-- This is a comment -->

03 HTML Attributes

üìù What Are Attributes?

  • Provide extra information about an element.
  • Written inside the opening tag.
  • Format: attribute="value"

Common Attributes

For Images (<img>)
  • src: Image source
  • width / height: Dimensions
  • alt: Description (for accessibility)
<img src="cat.jpg" width="300" height="200" alt="A cute cat">

For Links (<a>)
  • href: Link target URL
  • target: Where to open (e.g., "_blank" for new tab)
  • title: Tooltip text
<a href="https://example.com" target="_blank" title="Visit Example">Click here</a>

For Inputs
  • type: Input type (text, password, email, etc.)
  • value: Default value
  • placeholder: Hint text

üîñ Types of Attributes

1. Global Attributes (Any element): id, class, title.

2. Local Attributes (Specific elements): src, href.

3. Boolean Attributes (On/Off, no value): disabled, required, autofocus.

04 Text Formatting

Headings & Paragraphs

  • Headings: <h1> (most important) to <h6> (least important). Used for structure.
  • Paragraph: <p> defines a paragraph. Browser adds spacing.

Breaks & Rules

  • <br>: Single line break.
  • <hr>: Horizontal rule (thematic break).

Inline Text Formatting (Semantic)

Tag Meaning
<strong> Strong importance (Bold)
<em> Emphasis (Italic)
<u> Underline
<mark> Highlight
<sup> Superscript (x2)
<sub> Subscript (H2O)

Generic Inline Container

<span> is a non-semantic inline container used for styling parts of text. 

<p>This is <span class="highlight">special</span> text.</p>

05 Block vs Inline Elements

Block Elements Inline Elements
Start on a new line Do not start on a new line
Take full width available Take only needed space
Used for structure (e.g., div, p, h1) Used inside text (e.g., span, a, strong)
