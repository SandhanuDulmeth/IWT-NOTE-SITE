Introduction to the Internet, Web & HTML

Learn about the Internet, World Wide Web, HTML basics, document structure, and how the web works.

üåê What is the Internet?

  • A worldwide system of interconnected computer networks and electronic devices
  • Communicates using protocols called TCP/IP
  • Allows access to information and communication globally

üìú History of the Internet

  • 1969: ARPANET project connected computers at universities and U.S. defense
  • 1990s: ARPANET, NSFnet, and other networks formed the modern Internet

‚öôÔ∏è How the Internet Works

  • Networks are connected through routers and switches
  • Data is broken into small packets when travelling
  • A router forwards packets between networks
  • A switch connects devices within a single network

üî¢ IP Address

  • Each device has a unique IP address
  • Packets travel independently and are reassembled at the destination
  • Examples:
    • IPv4: 192.168.1.10
    • IPv6: 2001:0db8:85a3::8a2e:0370:7334

üì° Protocols & ISP

  • Protocols define communication rules
  • An ISP (Internet Service Provider) connects users to the global network

üåç World Wide Web (WWW)

  • A system of interlinked web pages written in HTML
  • Accessed using web browsers over the Internet
  • Built using: HTML, CSS, JavaScript, HTTP

üõ†Ô∏è Roles of Technologies

  • HTML: Structure of web pages
  • CSS: Layout and design
  • JavaScript: Interactivity
  • Pages are linked by hyperlinks

üñ•Ô∏è Web Components

  • Web servers: Store and deliver pages
  • Browsers: Request and display pages
  • HTTP: Protocol used for communication

üìÖ History of Web Development

  • 1989: Tim Berners-Lee proposed the Web
  • 1991: First website published
  • 1993: Mosaic graphical browser
  • Today: Billions of interactive sites

üîÑ Evolution of the Web

Web 1.0 (Read-only)

  • Static HTML pages ‚Äî no user interaction
  • Information publishing only

Web 2.0 (Read-write)

  • Interactive and participative ‚Äî users create content
  • Social media and dynamic pages
  • Technologies: AJAX, JavaScript frameworks

Web 3.0 (Semantic Web)

  • Machines understand data meaning
  • Metadata, linked data, AI and personalized recommendations

Web 4.0

  • Access anywhere via devices and IoT
  • Cloud computing, real-time communication

Web 5.0

  • Emotionally aware and highly personalized web
  • AI, machine learning, VR, brain-computer interfaces
  • Immersive experiences

üîÄ Internet vs World Wide Web

Internet World Wide Web
Physical network infrastructure Service using that infrastructure
Uses TCP/IP Uses HTTP/HTTPS
Includes email, chat, FTP, video calls Includes web pages and web apps
Connects computers Connects information

üîó URL (Uniform Resource Locator)

  • Identifies the location of a web resource
  • Shown in the browser address bar
  • Examples:
    • https://www.amazon.com/books (Commercial)
    • https://ucsc.cmb.ac.lk/about (Academic)
  • URL Structure: Protocol (scheme) ‚Üí Subdomain ‚Üí Second-level domain ‚Üí Top-level domain ‚Üí Path

üìá DNS (Domain Name System)

  • Converts domain names into IP addresses
  • Browser asks DNS for the IP before connecting to the server

üîÑ How the Web Works (Client‚ÄìServer Model)

  1. User enters a URL
  2. Browser sends a request
  3. DNS translates domain to IP
  4. Browser connects to the server
  5. Server responds with the website
  6. Browser displays the page

üìÑ Static vs Dynamic Web Pages

Static Web Pages

  • Built using HTML and CSS
  • Same content for all users (e.g., news sites)

Dynamic Web Pages ‚Äî Client-side

  • Use JavaScript / AJAX
  • Examples: form validation, interactive menus, games

Dynamic Web Pages ‚Äî Server-side

  • Use PHP, ASP, JSP, C#, SQL, etc.
  • Interact with databases ‚Äî store, retrieve, update, or delete data

üìù What is HTML?

  • HTML = HyperText Markup Language
  • Standard language for creating web page structure
  • Organizes text, images, and other content using markup tags

üìå HTML Versions

  • HTML 1.0 (1993) ‚Äì basic structure
  • HTML 4.01 (1999) ‚Äì more formatting
  • HTML5 (2014) ‚Äì modern, semantic, multimedia support

‚ú® HTML5 Features

  • Simple <!DOCTYPE html> declaration
  • New semantic tags: <header>, <footer>, <article>, <section>
  • Audio, video, canvas support
  • Improved forms
