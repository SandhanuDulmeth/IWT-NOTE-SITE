Introduction to PHP

Server-side scripting fundamentals ‚Äî what PHP is, how it works, syntax basics, variables, data types, operators, control structures, and loops.

01 Introduction to PHP

Lecture 1 ¬∑ SCS 1311 ¬∑ Topics: PHP overview, server-side scripting, client vs server

‚ö†Ô∏è What Problem Does PHP Solve?

Early websites were static ‚Äî same content for every user, no interaction, no data storage.

Modern dynamic websites need to:

  • Handle user input (forms, logins)
  • Work with databases
  • Generate different content for different users

üêò What is PHP?

  • A server-side scripting language for dynamic web apps
  • Originally: Personal Homepage ‚Üí now: PHP: Hypertext Preprocessor
  • Embedded inside HTML
  • Executed on the server before reaching the browser
  • Over 75% of websites use PHP

üñ•Ô∏è What Does "Server-Side" Mean?

  1. Code runs on the web server
  2. Browser sends a request
  3. Server processes PHP
  4. Server sends back only the result (HTML)

Users cannot see or modify PHP logic, making it more secure.

‚öñÔ∏è Client-Side vs Server-Side

Aspect Client-Side Server-Side
Runs in Browser Web server
Languages HTML, CSS, JS PHP, Java, Python
Speed Fast UI updates Depends on server
Security Less secure More secure

üåê Where PHP is Used

  • Dynamic websites & Content Management Systems
  • E-commerce systems & Web applications
  • Examples: WordPress, Facebook, Wikipedia

üóÑÔ∏è PHP and Databases

PHP can store, retrieve, update, and delete data. Common databases: MySQL, PostgreSQL, SQLite

‚öôÔ∏è How PHP Works (Step-by-Step)

  1. User enters URL
  2. Browser sends request to server
  3. Server finds the PHP file
  4. PHP executes on server
  5. PHP generates HTML
  6. HTML sent back to browser

A typical website: HTML ‚Üí structure, CSS ‚Üí styling, JavaScript ‚Üí interactivity, PHP ‚Üí logic & data, Database ‚Üí storage

03 Syntax & Basics

üìù PHP Syntax

üî§ Case Sensitivity

  • Variables ‚Üí case-sensitive ($name ‚â† $Name)
  • Keywords/functions ‚Üí not case-sensitive (ECHO = echo)

üì§ echo Statement

  • Outputs text, variables, or HTML to the browser
  • Does not return a value
  • Double quotes ‚Üí variables are replaced
  • Single quotes ‚Üí variables treated as plain text
php ‚Äî echo

$message = "Hello";

echo $message;              // Hello
echo "5x5=$number";       // 5x5=25  (variable replaced)
echo '5x5=$number';       // 5x5=$number  (literal text)

04 Variables & Data Types

üì¶ PHP Variables

  • Container to store data
  • Start with $
  • Type is detected automatically

üìè Naming Rules

  • Must start with $
  • Start with letter or underscore (not a number)
  • No spaces allowed
  • Case-sensitive

‚úÖ Valid: $studentName, $_totalMarks
‚ùå Invalid: $1name, $student name

üè∑Ô∏è PHP Data Types

Type Example
String "Kamal"
Integer 22
Float 5.6
Boolean true
php ‚Äî data types

$height = 5.6;         // Float
$isStudent = true;     // Boolean

echo "My name is $name";

üî≠ Variable Scope

  • Local ‚Äî declared inside a function, only accessible there
  • Global ‚Äî declared outside functions; use global keyword inside functions
  • Static ‚Äî keeps its value between function calls
Feature Global Static
Where defined? Outside function Inside function
Accessible everywhere? Yes (with global) No
Remembers value between calls? Yes (naturally) Yes
Scope Entire script Only inside that function
php ‚Äî variable scope 
// Local scope
function showValue() {
  $x = 10;
  echo $x;
}

// Global scope
$x = 5;
function test() {
  global $x;
  echo $x;
}

// Static variable
function counter() {
  static $count = 0;
  $count++;
  echo $count;
}

06 Control Structures

php ‚Äî if / if-else / if-elseif-else

  echo "Pass";
}

  echo "Pass";

  echo "Fail";

  echo "A";

  echo "B";

    echo "Weekday";
    break;
  case "Sunday":
    echo "Holiday";
    break;
  default:
    echo "Invalid day";
}

07 Loops

php ‚Äî while & do-while 
// while loop
$i = 1;
while ($i <= 5) {
  echo $i;
  $i++;
}

// do-while loop
$i = 1;
do {
  echo $i;
  $i++;
} while ($i <= 5);
php ‚Äî for & foreach 
// for loop
for ($i = 1; $i <= 5; $i++) {
  echo $i;
}

// foreach loop
$colors = array("Red", "Blue", "Green");
foreach ($colors as $color) {
  echo $color;
}

‚èπÔ∏è Break & Continue

  • break ‚Äî stops the loop completely
  • continue ‚Äî skips current iteration
php ‚Äî break & continue 
for ($i = 1; $i <= 5; $i++) {
  if ($i == 3) continue;  // skip 3
  if ($i == 5) break;     // stop at 5
  echo $i;  // outputs: 1 2 4
}
