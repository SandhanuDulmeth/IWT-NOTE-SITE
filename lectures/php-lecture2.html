01 Indexed Arrays

Lecture 2 ¬∑ SCS 1311 ¬∑ Topics: arrays, indexed arrays, creating, accessing, modifying

üì¶ What is an Array?

A collection of values stored under a single variable name, accessed using an index number or key name.

üóÇÔ∏è Three Types of Arrays

  1. Indexed ‚Äî numeric keys (0, 1, 2‚Ä¶)
  2. Associative ‚Äî named keys (key ‚áí value)
  3. Multidimensional ‚Äî arrays inside arrays

‚ùì Why Arrays Matter

  • Store multiple values in one variable
  • Can hold different data types
  • $_POST and $_GET are associative arrays for form data

üî¢ Creating Indexed Arrays

Numeric indexes start at 0. Used for ordered lists.

php ‚Äî creating indexed arrays 
// Method 1: array() syntax
$sports = array("Soccer", "Tennis", "Cricket", "Rugby");

// Method 2: Short syntax
$colors = ["Red", "Green", "Blue", "Yellow"];
// 0 => Red, 1 => Green, 2 => Blue, 3 => Yellow

üñ®Ô∏è Printing Arrays

Function Purpose
echo $arr Prints "Array" ‚Äî not useful for arrays
print_r($arr) Shows index and values ‚Äî good for debugging
var_dump($arr) Shows type, count, and string lengths
php ‚Äî access, modify, add, remove 
// Accessing elements
echo $sports[0];  // Soccer
echo $sports[2];  // Cricket

// Modifying elements
$sports[1] = "Badminton";  // replaces "Tennis"

// Adding elements
$sports[] = "Hockey";           // auto index
$sports[4] = "Hockey";          // explicit index
array_push($sports, "Volleyball", "Swimming");

// Removing elements
unset($sports[3]);    // removes element (index NOT re-ordered)
array_pop($sports);    // removes last element
php ‚Äî looping indexed arrays 
foreach ($sports as $sport) {
  echo $sport . "<br>";
}

02 Associative Arrays

üè∑Ô∏è Definition

Uses named keys instead of numbers. Stores data as key ‚áí value pairs. Common in form data and database records.

php ‚Äî associative arrays 
// Creating
$player = [
  "name"  => "Kamal",
  "sport" => "Cricket",
  "age"   => 22
];

// Accessing
echo $player["sport"];  // Cricket

// Modifying
$player["age"] = 23;

// Looping
foreach ($player as $key => $value) {
  echo $key . " : " . $value . "<br>";
}

üèãÔ∏è Practice

  1. Create an associative array for a student with name, age, and course
  2. Display all key-value pairs using foreach
  3. Modify one value and add a new key

03 Multidimensional Arrays

üß© Definition

An array containing other arrays. Used to represent tables or multiple records.

php ‚Äî multidimensional arrays 
// Creating
$players = [
  ["name" => "Kamal", "sport" => "Cricket"],
  ["name" => "Nimal", "sport" => "Rugby"]
];

// Accessing
echo $players[0]["name"];   // Kamal
echo $players[1]["sport"];  // Rugby

// Looping
foreach ($players as $player) {
  echo $player["name"] . " - " . $player["sport"] . "<br>";
}

üèãÔ∏è Practice

  1. Create a multidimensional array of 3 students with name, age, and course
  2. Loop through and display each student's info

04 Essential Array Functions

üîß Key Functions

Function Purpose
count($arr) Returns number of elements
isset($arr[key]) Checks if index/key exists and is not null
empty($arr) Checks if array has no data
php ‚Äî count, isset, empty 
// count()
echo count($sports);  // number of elements

// isset() ‚Äî prevents "Undefined index" errors
if (isset($sports[2])) {
  echo $sports[2];
}

// empty()
if (empty($sports)) {
  echo "Array is empty";
} else {
  echo "Array has values";
}

05 Array Operators

‚ûï Union Operator (+)

Combines arrays. Keeps values from the first array if keys match.

‚öñÔ∏è Comparison Operators

Operator Meaning
== Same key‚Äìvalue pairs (order doesn't matter)
=== Same pairs, same order, same types
!= / <> Arrays are different
!== Not identical (order or type differs)
php ‚Äî array operators 
// Union
$a = ["Soccer" => 1, "Tennis" => 2];
$b = ["Tennis" => 3, "Cricket" => 4];
$result = $a + $b;  // Tennis keeps value 2 from $a

// Comparison
$x = ["a" => 1, "b" => 2];
$y = ["b" => 2, "a" => 1];
$z = ["a" => 1, "b" => "2"];

var_dump($x == $y);    // true  ‚Äî same pairs
var_dump($x === $y);   // false ‚Äî different order
var_dump($x !== $z);   // true  ‚Äî different types

06 PHP Functions

üîß What is a Function?

A reusable block of code that helps reduce repetition, organize logic, and improve readability.

php ‚Äî creating & calling functions 
// Basic function
function displaySport() {
  echo "Soccer";
}
displaySport();  // Soccer
php ‚Äî parameters & return 
// With parameters
function showSport($sport) {
  echo "Selected sport: " . $sport;
}
showSport("Cricket");
showSport("Tennis");

// Returning values
function getSport($sport) {
  return "Selected sport: " . $sport;
}
$result = getSport("Rugby");
echo $result;
php ‚Äî default parameters 
function setTeamSize($size = 11) {
  return $size;
}

echo setTeamSize();    // 11  (uses default)
echo setTeamSize(7);   // 7   (overrides default)
